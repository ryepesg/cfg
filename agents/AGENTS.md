# Repository agent guidance

Keep this file specific to the repository it governs.

- State the repository's verification commands and trust boundaries.
- Keep secrets, credentials, private machine paths, and work-derived material
  out of standing instructions.
- Put reusable, task-specific procedures in `.agents/skills/<name>/SKILL.md`.
- Treat model instructions as guidance, not as a security or authorization
  boundary.

This template is copied into a repository. It is not globally delivered to a
harness home.
